# 吉他乐句知识库 - 设计文档

## 1. 项目概述

### 1.1 项目背景

构建一个本地桌面应用，用于存储和管理吉他五线谱乐句片段。支持以五线谱和吉他六线谱(Tab)两种形式展示乐句，提供标签分类、分析笔记和全文搜索功能，并具备MIDI音频播放能力。

### 1.2 核心价值

- 积累和管理个人吉他乐句素材库
- 通过标签和笔记快速检索乐句
- 可视化理解乐句在吉他指板上的位置
- 支持乐句播放，便于记忆和练习

## 3. 功能需求

### 3.1 核心功能列表

| 功能编号 | 功能名称 | 功能描述 | 优先级 |
|----------|----------|----------|--------|
| F1 | 乐句存储 | 以MusicXML格式存储五线谱乐句 | P0 |
| F2 | 五线谱展示 | VexFlow渲染五线谱，支持缩放 | P0 |
| F3 | 吉他Tab展示 | VexFlow渲染吉他六线谱，纯展示 | P0 |
| F4 | MIDI播放 | 软件合成播放乐句 | P0 |
| F5 | 标签管理 | 预设风格标签 + 自定义自由标签 | P0 |
| F6 | 分析笔记 | 文本笔记记录乐句特点和分析 | P0 |
| F7 | 标签筛选 | 通过点击标签筛选乐句 | P0 |
| F8 | 笔记搜索 | 全文检索笔记内容 | P0 |
| F9 | 组合搜索 | 标签筛选 + 笔记搜索同时生效 | P1 |

### 3.2 功能详细说明

#### F1 乐句存储
- 每个乐句以MusicXML格式存储
- 字段：id(UUID)、title、musicXml、createdAt、updatedAt、tags、note
- BPM仅用于播放控制，不保存在乐句数据中

#### F2 五线谱展示
- VexFlow渲染MusicXML内容
- 支持鼠标滚轮缩放
- 显示音符、节拍、调号

#### F3 吉他Tab展示
- 从MusicXML转换生成Tab表示
- 6条线对应吉他6根弦
- 纯展示，不可编辑

#### F4 MIDI播放
- Web MIDI API + SoundFont合成
- 播放控制：播放、暂停、停止
- 显示当前播放位置

#### F5 标签管理
**预设风格标签（8个）：**
- Blues（布鲁斯）
- Jazz（爵士）
- Rock（摇滚）
- Country（乡村）
- Funk（放克）
- Pop（流行）
- R&B（节奏布鲁斯）
- Soul（灵魂乐）

**自定义标签：**
- 用户可自由添加任意标签
- 标签自动补全

#### F6 分析笔记
- 富文本编辑器（非必须，最初版本为普通文本框）
- 记录乐句特点、和弦进行、演奏技巧等

#### F7 标签筛选
- 侧边栏显示所有使用的标签
- 点击标签筛选乐句列表
- 支持多标签组合筛选

#### F8 笔记搜索
- SQLite FTS5全文检索
- 实时搜索，输入即显示结果

#### F9 组合搜索
- 标签筛选 + 笔记搜索同时生效
- 搜索结果高亮匹配文本

## 5. 数据模型

### 5.1 SQLite表结构

```sql
-- 乐句主表
CREATE TABLE phrases (
    id TEXT PRIMARY KEY,
    title TEXT NOT NULL,
    music_xml TEXT NOT NULL,
    note TEXT DEFAULT '',
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL
);

-- 标签表
CREATE TABLE tags (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL UNIQUE,
    type TEXT NOT NULL CHECK(type IN ('preset', 'custom'))
);

-- 乐句-标签关联表
CREATE TABLE phrase_tags (
    phrase_id TEXT NOT NULL,
    tag_id TEXT NOT NULL,
    PRIMARY KEY (phrase_id, tag_id),
    FOREIGN KEY (phrase_id) REFERENCES phrases(id) ON DELETE CASCADE,
    FOREIGN KEY (tag_id) REFERENCES tags(id) ON DELETE CASCADE
);

-- 全文检索（使用LIKE实现，sql.js不支持FTS5）
-- SELECT * FROM phrases WHERE note LIKE '%keyword%' OR title LIKE '%keyword%';
```

### 5.2 数据流

```
用户输入MusicXML
      ↓
验证MusicXML格式
      ↓
VexFlow渲染预览 ←→ 用户编辑调整
      ↓
保存到SQLite
      ↓
VexFlow展示 + MIDI播放
```

## 7. 验收标准

### 7.1 功能验收

| 功能 | 验收条件 |
|------|----------|
| 乐句存储 | 可创建、编辑、删除乐句，数据持久化 |
| 五线谱展示 | MusicXML正确渲染为五线谱 |
| Tab展示 | 五线谱下方显示对应吉他Tab |
| MIDI播放 | 点击播放可听到正确旋律 |
| 标签管理 | 可添加预设/自定义标签，标签筛选生效 |
| 分析笔记 | 可编辑保存笔记 |
| 搜索 | 输入关键词可快速检索笔记内容 |

### 7.2 非功能验收

| 条件 | 标准 |
|------|------|
| 启动时间 | < 3秒 |
| 搜索响应 | < 100ms |
| 内存占用 | < 300MB |
| 数据库容量 | 支持1000+乐句 |
